# 프로젝트: 한국 공시 투자 판단 에이전트

## 목적
- 한국 주식 공시를 입력하면 내용 해석·영향도 판단·유사 사례·주가 패턴 분석을 통해 투자 액션(손절/익절/진입/관망)을 제시하고, 별도 챗봇으로 재무 데이터를 자연어로 즉시 조회할 수 있는 멀티에이전트 앱

## 사용 대상
- 공시를 읽어도 해석이 어려운 개인 투자자

## 유저 시나리오

### 기능 1 — 공시 투자 판단
1. 사용자가 종목명 또는 공시 URL을 입력하면
2. Supervisor가 에이전트를 순차·병렬 호출하여 공시 해석 → 영향도 판단 → 유사 사례 탐색 → 주가 패턴 분석을 수행하고
3. 결과로 "이 공시가 무슨 의미인지 + 과거 유사 사례 주가 흐름 + 손절/익절/진입/관망 의견"을 담은 리포트를 출력한다

### 기능 2 — 재무 데이터 챗봇
1. 사용자가 "삼성전자 10개년 매출액 가져와줘" 처럼 자연어로 입력하면
2. Query 에이전트가 의도를 파악해 DART API를 자율 호출하고
3. 결과로 연도별 데이터 표 + 차트를 채팅창에 즉시 출력한다

---

## 에이전트 구조

### 기능 1 — 공시 투자 판단

```
Supervisor
│
├── Step 1. ReAct Agent — 공시 해석 에이전트
│   ├── DART에서 공시 원문 수집
│   ├── 1단계: 공시 유형 분류 (룰 기반)
│   │   ├── 분석 대상: 유상증자, CB/BW, 잠정실적, 합병/분할,
│   │   │             최대주주변경, 대규모계약, 관리종목/상장폐지
│   │   └── 해당 없음 → "단순 정기공시" 조기 종료
│   ├── 2단계: 명세 기반 핵심 수치·항목 추출
│   │   └── 공시 유형별 추출 항목 정의에 따라 수치 파싱
│   │       (상세 기준 → disclosure_spec.md 참고)
│   └── 3단계: LLM 자율 판단
│       ├── 추출된 수치 + 공시 본문 맥락 종합
│       ├── 긍정 / 부정 / 중립 + 근거 서술
│       └── 최종 영향도: HIGH / MEDIUM / LOW
│           ├── LOW → 조기 종료
│           └── HIGH / MEDIUM → Step 2~4 진행
│
├── Step 2. ReAct Agent — 유사 공시 탐색 에이전트
│   ├── 동업종 + 동일 공시 유형 + 최근 3년 이내 조건으로 DART 검색
│   ├── 유사도 낮으면 검색 조건 완화하며 재탐색 루프
│   └── 유사 공시 발표 기업 리스트 반환
│
├── Step 3. ReAct Agent — 주가 패턴 분석 에이전트
│   ├── KRX에서 유사 공시 발표 기업들의 공시 전후 주가 수집
│   ├── 공시 후 1주 / 1개월 / 3개월 / 6개월 / 1년 수익률 계산
│   └── 평균 주가 흐름 패턴 + 분포 시각화
│
└── Step 4. Writer Agent — 투자 판단 리포트 작성
    ├── 공시 내용 요약 (쉬운 말로)
    ├── 영향도 판단 근거
    ├── 유사 사례 주가 흐름 요약
    └── 최종 액션 의견: 손절 / 익절 / 신규진입 / 관망
```

### 기능 2 — 재무 데이터 챗봇

```
ReAct Agent — Query 에이전트
├── 자연어 파싱 → 종목명 / 지표명 / 기간 추출
├── DART API 파라미터 자율 결정 후 호출
├── 데이터 부족 or 오류 시 재시도 루프
└── 표 + 차트 렌더링 후 반환
```

## 공시 영향도 판단 프로세스 요약

```
공시 입력
↓
[1단계] 유형 분류 (룰 기반) → 대상 외 공시 조기 종료
↓
[2단계] 핵심 수치·항목 추출 (명세 기반) → disclosure_spec.md
↓
[3단계] LLM 자율 판단 (맥락 + 수치 종합)
↓
HIGH / MEDIUM → 유사사례 + 주가패턴 + 액션 의견
LOW → 조기 종료
```

## 활용 기술

| 항목 | 내용 |
|------|------|
| **프레임워크** | LangGraph StateGraph |
| **패턴** | Supervisor + ReAct Agent |
| **LLM** | GPT-4o / Claude Sonnet (공시 해석, 영향도 판단) |
| **UI** | Streamlit (탭 분리: 공시 판단 / 데이터 챗봇) |

### Tools

| Tool | 용도 |
|------|------|
| `dart_disclosure_fetch` | DART — 공시 원문 수집 |
| `dart_disclosure_search` | DART — 유사 공시 탐색 |
| `dart_financials` | DART — 재무제표 조회 (챗봇) |
| `krx_price` | KRX — 주가 수집 및 수익률 계산 |
| `web_search` | 웹 검색 — 컨센서스·업종 맥락 보완 |
| `chart_renderer` | plotly — 주가 패턴·재무 데이터 시각화 |

### API

- **DART OpenAPI** (`opendart.fss.or.kr`) — 공시 원문, 유사 공시 탐색, 재무제표
- **KRX 시세 API** (`data.krx.co.kr`) — 주가 수집
- **웹 검색** — Tavily Search (컨센서스, 업종 맥락)




# 공시 유형별 판단 기준 명세 (disclosure_spec.md)

> 이 문서는 공시 해석 에이전트의 2단계(명세 기반 추출)와 3단계(LLM 자율 판단) 설계 기준이다.
> 명세 기반 추출은 파싱할 항목을 고정하고, LLM 판단은 추출된 수치를 맥락과 종합해 자율 해석한다.

---

## 1. 잠정실적

### 명세 기반 추출 항목
- 매출액·영업이익·순이익 (당기 실적 수치)
- YoY 증감률 (%)
- 컨센서스 수치 (웹 검색으로 별도 수집)
- 컨센서스 대비 괴리율 (%)
- 일회성 항목 여부 (자산매각익, 환율효과, 충당금 등)

### LLM 판단 지시
- 괴리율 수치만이 아니라 일회성 항목 제거 후 실질 실적인지 판단
- 매출은 좋은데 이익이 나쁜 경우 비용 구조 문제인지 해석
- 컨센서스 자체가 이미 낮게 잡혀있던 건지 시장 기대 맥락 파악
- 서프라이즈 / 쇼크 / 중립 판단 + 근거 서술

---

## 2. 유상증자

### 명세 기반 추출 항목
- 증자 목적 (시설투자 / 운영자금 / 채무상환 / 타법인 취득)
- 희석률 (신주 발행 수 / 기존 발행 주식 수 %)
- 발행가액 할인율 (기준주가 대비 %)
- 배정 방식 (주주배정 / 제3자배정 / 일반공모)
- 납입일, 신주 상장 예정일

### LLM 판단 지시
- 목적이 진짜 성장을 위한 건지, 급한 자금 조달인지 맥락 파악
- 제3자배정이면 인수 주체 성격 판단 (전략적투자자 vs 불명확한 주체)
- 희석률 + 할인율 조합으로 기존 주주 피해 정도 종합 판단
- 현재 주가 수준 대비 발행가 타이밍 해석
- 긍정 / 부정 / 중립 판단 + 근거 서술

---

## 3. 전환사채(CB) / 신주인수권부사채(BW)

### 명세 기반 추출 항목
- 발행 규모 (자기자본 대비 %)
- 전환가액 (현재 주가 대비 프리미엄/디스카운트 %)
- 리픽싱 조항 유무 + 하한선
- 만기, 표면이자율, 풋옵션(조기상환청구권) 조건
- 인수자 정보 (기관 / 개인 / 외국계 / 사모펀드)

### LLM 판단 지시
- 리픽싱 조항 있으면 주가 하락 시 전환가 낮아져 추가 희석 발생 → 오버행 리스크 판단
- 인수자 성격으로 자금 목적 추론 (사모펀드면 엑싯 목적 가능성)
- 발행 규모 + 현재 재무상태 종합해서 재무 악화 신호인지 판단
- 전환가 대비 현재 주가 위치로 실질 희석 시점 추론
- 긍정 / 부정 / 중립 판단 + 근거 서술

---

## 4. 합병 / 분할

### 명세 기반 추출 항목
- 합병 비율
- 상대 기업 매출·영업이익·부채 규모
- 합병 목적 (수직계열화 / 사업다각화 / 구조조정)
- 합병 방식 (흡수합병 / 신설합병)
- 소규모합병 여부 (주주총회 생략 가능 여부)

### LLM 판단 지시
- 상대 기업이 알짜인지 부실인지 재무 수치로 판단
- 합병 목적이 진짜 시너지인지 부실 떠안기인지 맥락 해석
- 합병 비율이 현재 주가 기준으로 유리한지 불리한지 판단
- 소규모합병이면 주주 의사결정 배제 여부 리스크 언급
- 긍정 / 부정 / 중립 판단 + 근거 서술

---

## 5. 최대주주변경

### 명세 기반 추출 항목
- 변경 전/후 최대주주명 및 지분율
- 취득 방법 (장내매수 / 블록딜 / 계약 양수도)
- 취득 단가 (현재 주가 대비 프리미엄/디스카운트 %)
- 인수 목적 기재 내용

### LLM 판단 지시
- 새 최대주주 성격 파악 (산업 내 전략적투자자 / PE / 개인 / 외국계)
- 취득 단가가 프리미엄이면 경영권 확보 의지, 디스카운트면 의도 불명확으로 해석
- 기존 최대주주가 왜 파는지 맥락 추론 (경영 실패 / 엑싯 / 재무 문제)
- 새 주주의 과거 투자 이력 기반으로 경영 개선 가능성 판단
- 긍정 / 부정 / 중립 판단 + 근거 서술

---

## 6. 대규모계약 체결

### 명세 기반 추출 항목
- 계약금액 (매출액 대비 %)
- 계약 상대방 (국내/해외, 기업 규모)
- 계약 기간
- 계약 유형 (단발성 / 장기공급 / 프레임워크)
- 선수금 여부

### LLM 판단 지시
- 계약 상대방 신뢰도 및 규모로 실현 가능성 판단
- 단발성인지 장기 매출로 이어지는지 해석
- 매출 인식 시점 (언제부터 실적에 반영되는지) 추론
- 계약금액 대비 실제 이익 기여도 추정
- 긍정 / 부정 / 중립 판단 + 근거 서술

---

## 7. 관리종목 / 상장폐지 지정

### 명세 기반 추출 항목
- 지정 사유 (매출액 미달 / 자본잠식 / 감사의견 / 거래량 미달)
- 지정 유예 기간
- 이의신청 가능 여부
- 현재 자본잠식률 (%)

### LLM 판단 지시
- 지정 사유가 일시적인지 구조적인지 판단
- 유예 기간 내 해소 가능성 추론 (자본 확충 가능성, 실적 회복 여부)
- 상장폐지까지 이어질 리스크 수준 판단
- 긍정 / 부정 / 중립 판단 + 근거 서술

---

## 최종 영향도 분류 기준

| 영향도 | 조건 | 후속 처리 |
|--------|------|----------|
| **HIGH** | LLM이 강한 긍정 or 부정으로 판단, 수치 근거 명확 | Step 2~4 전체 실행 |
| **MEDIUM** | LLM이 중립 or 판단 애매, 맥락 불충분 | Step 4만 실행 (간략 해석 + 의견) |
| **LOW** | 분석 대상 공시 유형 아님 or 영향 미미 | 조기 종료 |
